Vector Divergence and Curl

Divergence measures how a vector field spreads out from a point; curl measures rotation. Input tip: you can use x^2 (caret) ‚Äî the app will accept it and convert to the correct Python/SymPy power operator.

Parameter Guide (Quick)

Fx, Fy, Fz: Enter expressions using x, y, z.

  • Use ^ for powers (e.g., x^2, y^2*z).
  • You can use functions: sin(), cos(), exp(), sqrt(), etc.
  • Examples:
    x^2 + y
    -x*y
    sin(x) + z^2

Point: Give comma-separated values x,y,z. If you give only x,y, then z=0 is assumed.

1,2,3 ‚Üí x=1, y=2, z=3
1,2 ‚Üí x=1, y=2, z=0

Plot span: How far around the point to draw the quiver plot. Default is 2. Increase it for a larger view.
